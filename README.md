# Recruitment Dashboard - Power BI
## Overview
This project analyzes recruitent data including applications, shortlisted candidates, and conversion funnel.
## Tools used
1) Power BI
2) Excel

#Features
1) Total Applications KPI
2) Shortlisted candidates
3) Funnel analysis
4) Domain wise Analysis
5) Location wise Analysis.
